環境構築
=======================================================

* Vagrant
    - 仮想環境の雛形を作成し、どこでも簡単に同じ環境を再現できるようにするもの
    - http://www.vagrantup.com/
* Docker
    - コンテナ技術を使った軽量な仮想環境
    - http://www.docker.com/
* Chef
    - 自動的にユーザーの作成やパッケージのインストール、設定ファイルの編集などを行うもの
    - http://www.getchef.com/
* VirtualBox
    - 無料な仮想環境
    - https://www.virtualbox.org/
